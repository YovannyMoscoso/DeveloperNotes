# Remote, Origin, and Main

## What is a Remote?

A remote is a repository hosted elsewhere.

Usually GitHub.

## What is Origin?

Origin is the default nickname for the GitHub repository.

Example:

origin
↓
github.com/user/project

## What is Main?

Main is the default branch.

It usually contains stable code.

## Example

Push local main branch:

git push origin main

Meaning:

Push
↓
to remote called origin
↓
using branch called main
