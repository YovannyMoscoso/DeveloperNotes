# Git Basics

## What is Git?

Git is a version control system.

It tracks changes made to files over time.

## Common Commands

Check status:

git status

Add files:

git add .

Commit changes:

git commit -m "message"

View history:

git log

## Important Concepts

Repository
- Project tracked by Git

Commit
- Snapshot of project state

Branch
- Independent line of development

Merge
- Combine branches

Remote
- Repository hosted elsewhere
