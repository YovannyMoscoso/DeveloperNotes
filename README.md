# DeveloperNotes

Personal notes for setting up development projects using Git, GitHub, Python, Virtual Environments, pip, pytest, and GitHub Actions.

## Purpose

This repository serves as my personal developer handbook.

The goal is to document:

- How to start a new project
- How Git and GitHub work
- How to configure Python environments
- How package management works
- How automated testing works
- How GitHub Actions and CI/CD work
- Common problems and solutions

This documentation helps me quickly set up new projects without repeatedly searching for the same information.
