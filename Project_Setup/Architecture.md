## Purpose of Each Tool

### Git

Tracks code changes locally.

### GitHub

Stores code remotely and allows collaboration.

### Virtual Environment

Creates an isolated Python environment for a specific project.

### pip

Installs and manages Python packages.

### requirements.txt

Stores project dependencies.

### pytest

Runs automated tests.

### GitHub Actions

Automates workflows.

### CI/CD

Automates validation and deployment processes.
